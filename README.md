# Cake Ordering System Design
Software Engineering Project

##  Project Overview

This project presents the analysis and design of a Cake Ordering System that allows customers to customize and order cakes online.

The system focuses on improving communication between customers and cake shop owners and reducing ordering errors.

---

##  System Analysis

The project includes:

- Problem definition
- Stakeholder analysis
- Functional requirements
- Non-functional requirements
- Feasibility study
- Project planning

---

##  System Design

The system design includes:

- Use Case Diagram
- Context Diagram
- Event Table
- Sequence Diagrams
- Class Diagram
